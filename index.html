Instructions

Set your /etc/opkg.conf accordingly:

dest root /
dest ram /tmp
lists_dir ext /var/opkg-lists
option overlay_root /overlay

Some packages don't work very well without using image from this site, as ext4 image has been generated in a different way in stock OpenWrt RPI image.
For example it doesn't allow resizing. Recommended way, would be to use image from this site.
Other issues might also occur in some cases.

Changes in rpi-openwrt's image

  • Patches against base system
  • dnsmasq and firewall are not included (can be installed as modules)
  • nano is installed
  • rpi-openwrt packages added to package repositories list
